<h1>**Arrow functions**</h1>

O que é?

Uma forma diferente de escrever função em JS

Características:

* Possui uma sintaxe mais curta.

* Ela otimiza a escrita do seu código, deixando-o mais limpo, enxuto e aumentando a legibilidade.

*Não tem seu próprio this, arguments, super ou new.target.

*São melhor aplicadas para funções que não sejam métodos.

*Não podem ser usadas como construtoras (constructors).

*Uma arrow function não pode conter uma quebra de linha entre seus parâmetros e sua flecha.

Vantagem:

Quando uma função tem só uma instrução e ela é de retorno, na função Arrow functions pode-se apagar o return.

Concise body: Quando o return não aparece.

Block body: Precisa do return declarado.

        Ex:

                Função normal:

                        function subtracao(a, b) {

         return a - b;

}

Arrow functions:

                        const subtracaoVersao2 = (a, b) => a - b;

 A Arrow functions é criada como se fosse criar uma constante, por isso usamos o “const”, ou seja, ele não usa a palavra chave “functions” para poder criar uma função, além disso, utiliza o => para definir a instrução da função e o valor que ela irá retornar.

Quando a função tiver só um parâmetro(instrução) não precisa de parentes

        Ex:

                const somar2Versao2 = a => a + 2;

Quando a função tiver 0 parâmetros(instrução)  precisa-se colocar parentes com nada dentro:

        Ex:

                const diaDoMesVersao2 = () => new Date().getDate();

                

new Date() -  cria uma informação de data e acessa com o dia atual, com o método getDate

Em funções de mais de um parâmetro e várias instruções, não tem muita diferença de uma função normal .

        Ex:

                Função normal:

                        function superFuncao(a, b) {         

let subtracao = a - b;

subtracao = subtracao - 2;         

let diaDoMes = new Date().getDate();         

return diaDoMes;

}

                Arrow functions:

const superFuncaoVersao2 = (a, b) => {         

let subtracao = a - b;

subtracao = subtracao - 2;         

let diaDoMes = new Date().getDate();         

return diaDoMes;

}
